⌨️ Command Reference

Complete List of Game Commands

Movement Commands

Command Description
north / n Move north
south / s Move south
east / e Move east
west / w Move west
up / u Go up (stairs, ladders)
down / d Go down

Information Commands

Command Description
look View current room description
examine [object] Look closely at an item or feature
inventory / inv View your items
score View your character stats
who See all players currently online in the game
help Show available commands
help [command] Get info on specific command

Combat Commands

Command Description
attack [target] Attack a monster or player
fight [target] Same as attack
kill [target] Same as attack
cast [spell] at [target] Cast a spell on target
kick [target] Attack with kick verb
slash [target] Attack with slash verb

Item Commands

Command Description
get [item] Pick up an item
drop [item] Drop an item from inventory
use [item] Use a consumable item
equip [item] Equip weapon/armor/shield
buy [item] Purchase from NPC shop
buy [item] from [npc] Buy specific item from NPC

Communication Commands

Command Description
say [message] Talk to everyone in room
tell [player] [message] Private message to player
talk to [npc] Speak with an NPC
ask [npc] about [topic] Ask NPC about something
greet [npc] Greet an NPC

Party Commands

Command Description
party create Create a new party
party invite [player] Invite player to your party
party accept [player] Accept party invitation
party leave Leave current party
party kick [player] Remove player from party (leader only)
party info View party members and stats

Quest Commands

Command Description
quests View all active quests
quest [name] View specific quest details
quest progress Check quest progress
quest abandon [name] Abandon a quest

Guild Commands

Command Description
guild create [name] Create a new guild (costs gold)
guild invite [player] Invite player to guild
guild accept [name] Accept guild invitation
guild info View guild information
guild leave Leave your guild
guild members List all guild members

Spell Commands

Command Description
spells List all known spells
cast [spell] Cast spell on yourself
cast [spell] at [target] Cast spell on target
memorize [spell] Learn a new spell (from book)

Utility Commands

Command Description
alias [shortcut] [command] Create a command shortcut
aliases List all your aliases
unalias [shortcut] Remove an alias
time Show current game time
weather Check current weather
rest Rest to recover HP/MP
save Save your progress (auto-saves)
quit Exit the game

💡 Command Tips

  • Commands are case-insensitive (NORTH = north)
  • Most commands have shortened versions (n for north)
  • Natural language works! "go north" = "north"
  • Use TAB to auto-complete (if available)
  • Press UP arrow to repeat previous commands
  • Commands parse context - "attack" knows to target nearby monsters
  • You can chain actions with semicolons: get sword; equip sword
